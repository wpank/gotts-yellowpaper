# Gotts: Permissionless Banking Infrastructure for Agent Capital Markets

**Yellow Paper — February 2026**

[gotts.ai](https://gotts.ai)

## Abstract

Autonomous agents now generate the majority of on-chain transaction volume across major decentralized exchanges, yet the financial infrastructure they depend on was designed for human operators. This paper presents Gotts, a permissionless protocol for autonomous agent capital management that formally integrates ERC-4626 tokenized vaults, ERC-8004 agent identity, and Uniswap V4 hooks into a co-evolutionary system.

The protocol makes six contributions: (i) a vault factory where any ERC-8004-registered agent can deploy capital vehicles with auto-created V4 share pools and auction-managed strategy rights; (ii) a cybernetic learning architecture grounded in the CoALA cognitive framework, Reflexion, and ExpeL, adapted for non-stationary financial environments; (iii) a layered safety architecture organized into preventive, cryptographic, and reactive rings, where three layers provide cryptographic enforcement that a fully compromised language model cannot bypass; (iv) a time-delayed execution proxy as the primary security primitive, motivated by the demonstrated vulnerability of all current trusted execution environments to sub-$50 hardware attacks; (v) reputation-weighted economics where custody quality, on-chain reputation, and learned strategy co-evolve through a formally specified feedback loop; and (vi) a permissionless executor framework with Dutch auction liveness guarantees.
